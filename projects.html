Machine Learning – Simulating Evolutionary Techniques to Create Images

During this project, I used machine learning and evolutionary techniques to recreate three images to the highest possible accuracy while using just 100 polygons. This project was a large success, as I achieved accuracy of 96% in all of these recreations. This was the highest accuracy achieved in my entire year group.

The algorithm mimics genetic evolution by iteratively mutating a population of candidate solutions, evaluating their fitness, and breeding the most successful candidates. It learns how to recreate input images using just a small set of geometric shapes.

I decided to create algorithms for three different evolutionary techniques in order to find the most efficient one. The algorithms I created were for random mutation, tournament selection, and roulette selection. I evaluated their speeds by constructing a graph using matplotlib

I used evolutionary techniques such as tournament selection, weighted roulette selection and colour mutation, as well as changing initial parameters in order to optimise the final results. A progression of accuracy based on these factors is shown below.

Data Structures & Algorithms Simulator

This project was developed in my own time, as I looked to improve my programming skills when working with various data structures and algorithms. The program visualises a wide variety of searching, sorting and traversal algorithms, allowing the user to customise the size of the dataset and speed of the algorithms.

  • Sorting algorithms: Selection Sort, Insertion Sort, Bubble Sort, Quick Sort, Merge Sort
  • Searching algorithms: Binary Search, Linear Search
  • Traversal algorithms: Depth-first Search, Breadth-first Search

A basic TKinter Python interface is provided to allow users to select their desired algorithm and parameters, then watch the algorithm in action! All algorithms are built from scratch based on the skills I've learnt when practicing Leetcode and other problems, and each is displayed using Matplotlib, a graph plotting library built for Python. HTCondor Job Interface

This project provides an easy to use front-end interface for HTCondor, allowing users to submit their deep learning jobs and other projects onto a HTCondor computing cluster without needing to run it all from the command line. This simplifies the process of running variations of jobs and different projects, allowing more time to be spent experimenting and doing the important work, along with opening up the possibility for non-computing experts and researchers to use HTCondor.

I was nominated to undertake this project by my university, as one of 24 interns from across the Northern 8 Universities, funded by the N8 CIR. This work was then presented in York to numerous experts and students from around the country.
